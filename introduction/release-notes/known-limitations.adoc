Table 3. KIE Tools

|===

|Limitation | Description
|Form Generation- Issue in navigating to the Forms view| For Quarkus-based https://incubator.apache.org/projects/kie.html[Apache Kie, window=_blank] projects, when you are using or testing Form by using the *jBPM Quarkus DevUI* extension, `runtime-tools-quarkus-extension` may not work properly after quarkus3 migration. With the quarkus3 migration, the UI of the `runtime-tools-quarkus-extension` is renewed and this issue blocks the users from navigating to the Forms view. You can access it on `http://localhost:8080/q/dev-ui/extensions` once you started mvn clean `quarkus:dev` in your project. For more information see https://github.com/apache/incubator-kie-issues/issues/836[issue,window=_blank].
